# 🎨 Prompt Mestre UI/UX Design Premium
## Projeto: Case-Já | Luxury Wedding SaaS

---

# Objetivo

Redesenhar completamente a interface do SaaS utilizando um conceito **Luxury Minimalism**, inspirado nas maiores marcas premium do mercado (Rolls-Royce, Four Seasons, Aston Martin, Vogue, Cartier, Ritz Paris, Dior e Apple).

O resultado deve transmitir:

- Exclusividade
- Sofisticação
- Elegância
- Alto padrão
- Experiência Premium
- Microinterações refinadas
- Visual cinematográfico
- Interface limpa
- Excelente UX

Jamais utilizar aparência genérica de dashboard administrativo.

---

# Conceito Visual

Criar uma experiência semelhante a um site de uma empresa de luxo.

A tela deve parecer uma revista premium.

Poucos elementos.

Muito espaço negativo.

Hierarquia visual impecável.

Interface extremamente elegante.
